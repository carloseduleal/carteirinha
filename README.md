## CARTEIRINHA

## INSTALL DEPENDENCIES

```
npm install
```

## RUNNING LOCAL SERVER

### Start mongodb
```
mongod --dbpath=/db/data
```

### Start node SERVER
```
npm start
```

### Testing
```
npm test
```
